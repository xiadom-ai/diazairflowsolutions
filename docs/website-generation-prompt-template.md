## Prompt Optimization Tips

### 1. Be Specific About Components
```markdown
Instead of: "Add testimonials"
Use: "Add testimonials section using carousel variant, pulling from Google Reviews API, minimum 4.5 stars, displaying 6 reviews with author name, rating, and date"
```

### 2. Define Interaction Patterns
```markdown
Specify:
- Hover effects: "Scale 1.05 on hover with 200ms ease-out"
- Scroll animations: "Fade in from bottom when 30% visible"
- Loading states: "Skeleton screens for dynamic content"
- Micro-interactions: "Button ripple effect on click"
```

### 3. Clarify Responsive Behavior
```markdown
Mobile (< 768px):
- Hamburger menu with slide-out drawer
- Stack columns vertically
- Increase touch targets to 44px minimum
- Hide decorative elements

Desktop (> 1024px):
- Sticky navigation with transparency
- Multi-column layouts
- Hover states enabled
- Parallax scrolling effects
```

### 4. Specify Content Hierarchy
```markdown
Hero Section:
- H1: Main value proposition (60 characters max)
- Subheading: Supporting statement (120 characters max)
- Primary CTA: Action verb + benefit
- Secondary CTA: Alternative action
```

### 5. Define Performance Constraints
```markdown
Critical Performance Metrics:
- First Contentful Paint: < 1.5s
- Largest Contentful Paint: < 2.5s
- Time to Interactive: < 3.5s
- Cumulative Layout Shift: < 0.1
- Bundle size: < 200KB initial JS
```

## Advanced Prompt Modifiers

### For Maximum Accessibility
```markdown
Accessibility Priority:
- Implement skip navigation links
- Ensure all interactive elements are keyboard accessible
- Add live regions for dynamic content updates
- Include focus trap for modals
- Implement proper heading hierarchy
- Add descriptive link text (no "click here")
- Ensure 4.5:1 color contrast minimum
```

### For SEO Optimization
```markdown
SEO Requirements:
- Implement dynamic meta tags for all pages
- Add structured data for [Organization/LocalBusiness/Product]
- Create XML sitemap with priority values
- Implement breadcrumbs with schema markup
- Add Open Graph and Twitter Card tags
- Include canonical URLs
- Implement hreflang for multi-language
```

### For Conversion Optimization
```markdown
CRO Elements:
- Add trust signals above fold (certifications, reviews, guarantees)
- Implement scarcity indicators (limited spots, countdown)
- Use contrasting colors for CTAs
- Add social proof notifications (recent purchases/signups)
- Include risk reversal (money-back guarantee, free trial)
- Implement progressive disclosure for forms
- Add exit-intent popup with special offer
```

### For Performance Enhancement
```markdown
Performance Optimizations:
- Implement route prefetching for Next.js
- Use dynamic imports for below-fold components
- Optimize images with next/image and sharp
- Implement service worker for offline capability
- Use CSS containment for layout stability
- Implement virtual scrolling for long lists
- Add resource hints (preconnect, prefetch, preload)
```
